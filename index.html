Choose what you’re doing

One person creates the room. Everyone else joins using the four-digit code.

Setting up? Choose the game and number of cards, then share the code. Joining? Enter your name and the code you were given.

Join someone else’s game

Ask the person who set up the game for their four-digit room code.

Set up a new game

Choose the settings, create the room, then send the code to everyone playing.

Alcohol is optional. Players can always choose a soft drink, skip a challenge or stop playing.

How a game works

There is no separate host screen — everyone plays on their own phone or tablet.

👑 Person setting up

1Tap Set Up a Game and choose the settings.
2Create the room and share the four-digit code.
3Wait for everyone to join, then tap Ready / Start.

🎟 Everyone joining

1Tap Join With a Code.
2Enter your name and the four-digit code.
3Wait in the room. Your bingo cards appear when the game starts.

When a called number is on your card, tap it to place a virtual dabber.
